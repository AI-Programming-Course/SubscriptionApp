# Subscription Tracker

A powerful desktop application for tracking and managing your subscriptions with comprehensive expense analytics, notifications, multi-currency support, and budget management.

## Features

- **Subscription Management**: Track daily, weekly, monthly, quarterly, yearly, and custom billing cycles
- **Expense Analytics**: Visual charts, spending trends, category breakdowns
- **Desktop Notifications**: Automatic reminders before subscription renewals
- **Multi-Currency Support**: Support for 150+ currencies with automatic exchange rates
- **Budget Management**: Set limits, track spending, get alerts
- **Secure**: Optional data encryption, secure architecture
- **Cross-Platform**: Works on Windows, macOS, and Linux

## Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm start
   ```

## Development

- `npm start` - Start the application
- `npm run dev` - Start in development mode
- `npm run build` - Build for all platforms
- `npm run build:mac` - Build for macOS
- `npm run build:win` - Build for Windows
- `npm run build:linux` - Build for Linux

## Tech Stack

- Electron 28
- Vanilla JavaScript (ES6+)
- Chart.js for analytics visualization
- localStorage for data persistence
